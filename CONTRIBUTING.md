# Contributing

Thank you for your interest in contributing to Hash3r.

We welcome:

- bug fixes
- documentation improvements
- feature enhancements
- algorithm additions
- test coverage

## Setup

```bash
git clone https://github.com/anku0669/Hash3r.git
cd Hash3r
pip install -r requirements.txt
```

## Workflow

1. Fork the repository
2. Create a feature branch
3. Commit your changes
4. Open a pull request

Please keep code modular, readable, and documented.
